SOFTWARE LIBRE.


¿Que es "software libre"?

Software libre es un programa informático donde el usuario propietario del programa tiene la libertad de copiarlo, modificarlo, redistribuirlo y distribuirlo para el beneficio de una comunidad. Software libre no significa que sea gratuito aunque también pueden serlo. Software libre viene de libertad o ‘libertad de expresión’ y conlleva 4 libertades fundamentales para considerarse un software libre: libertad de ejecutar el programa no importanto el propósito que se tenga. libertad de acceso al código fuente: esto significa la posibilidad de estudiar y modificar el programa. libertad de redistribuir el programa y libertad de distribuir copias de versiones modificadas. Si la ejecución o uso de un software que se considera libre está conectado a la ejecución de otro que no lo es entonces no se debe considerarse libre sino una práctica de engaño llamado “tivoización”, “bloqueo” o llamado por las empresas interesadas como “arranque seguro”. Los softwares libres son asociados generalmente a la gratuidad pero la comercialización de softwares libres también es común y cumple un rol importante en la mejora sistematizada de programas.



Ejemplos de software libre:

Los softwares libres también se caracterizan o se conocen porque pueden ser descargadas, generalmente gratuitamente, a través de Internet. Algunos ejemplos de software libres son:
1. LINUX: sistema operativo
2. Open Office: análogo a los programas Office de Windows que es software propietario.
3. Gimp: editor de imágenes
4. Ubuntu: sistema operativo
5. VLC: reproductor multimedia
6. Jquery: librería quefacilita la programación del lenguaje Javascript.






VENTAJAS DE SOFTWARE LIBRE.

1. Costo: Generalmente es gratuito, lo que reduce significativamente los costos en comparación con software propietario.
2. Libertad: Permite a los usuarios ejecutar, estudiar, modificar y distribuir el software, lo que fomenta la personalización y adaptación a necesidades específicas.
3. Transparencia: El acceso al código fuente permite una mayor transparencia y confianza en el software, ya que los usuarios pueden auditar y verificar su seguridad y funcionamiento.
4. Comunidad: Suelen tener comunidades activas que contribuyen al desarrollo y soporte, proporcionando una fuente de innovación y mejora continua.
5. Flexibilidad y Adaptabilidad: Al poder modificar el código, se puede adaptar el software a entornos específicos o necesidades cambiantes sin esperar a que el proveedor lance una nueva versión.
6. Independencia del Proveedor: No depende de un único proveedor para actualizaciones, soporte o mejoras, lo que puede ser beneficioso para la continuidad del negocio.
7. Seguridad: Las vulnerabilidades pueden ser detectadas y corregidas más rápidamente debido al acceso abierto al código y a la colaboración comunitaria.



DESVENTAJAS DE SOFTWARE LIBRE.

1. Soporte: El soporte oficial puede ser limitado o inexistente, dependiendo del proyecto, lo que puede requerir personal técnico capacitado o recurrir a servicios externos.
2. Curva de Aprendizaje: Puede ser más difícil de usar y requerir una curva de aprendizaje más pronunciada, especialmente para usuarios no técnicos.
3. Compatibilidad: A veces puede haber problemas de compatibilidad con otros software o hardware propietarios.
4. Documentación: La documentación puede ser insuficiente o desorganizada, lo que dificulta la resolución de problemas o la implementación del software.
5. Calidad y Estabilidad: No todos los proyectos de software libre tienen la misma calidad y estabilidad, lo que puede llevar a inconsistencias o problemas en su uso.
6. Seguridad: Aunque la transparencia es una ventaja, también puede ser una desventaja si las vulnerabilidades no son detectadas y explotadas por actores maliciosos antes de ser corregidas.
7. Requiere Recursos Técnicos: La personalización y mantenimiento pueden requerir recursos técnicos que algunas organizaciones no tienen disponibles.





KDE.


¿Que es KDE?

KDE, es uno de los entornos de escritorio más populares para el sistema operativo Linux. Cuenta con una interesante interfaz, muy amigable y atractiva para usuarios que migran de otros sistemas operativos como Windows, siendo una de las características principales por las que es uno de los entornos de escritorio más populares en las distribuciones Linux.




ENTORNO DE ESCRITORIO.

Un entorno de escritorio, es un software que proporciona al usuario un área de trabajo completamente visual para poder interactuar de manera cómoda e intuitiva con el ordenador.
Para lograr este objetivo, los entornos de escritorio utilizan diversas herramientas, como iconos, ventanas, animaciones y sistemas de menú desplegables que interactúan entre sí, para ofrecer la mayor comodidad posible al usuario final de un sistema operativo.
Cabe destacar, que los entornos de escritorio se integran perfectamente con dispositivos de entrada y salida de datos, como teclados, mouse, monitor, entro otros, para mantener una comunicación fluida con el sistema.


KDE.

KDE, es un proyecto internacional de software libre, orientado específicamente a desarrollar un entorno de escritorio e infraestructura de desarrollo para los sistemas operativos GNU/Linux, Mac OSX e incluso, Windows.
KDE es un entorno de escritorio moderno, que ofrece un extraordinario soporte para hasta 75 idiomas diferentes, lo que ofrece a los usuarios un nivel de comprensión del entorno mucho más adaptado a su idioma.
Una de las ventajas de KDE, es que está hecho para ser personalizado. Es por esta razón, que viene con una gran cantidad de funciones específicas que ofrecen la posibilidad de adaptar el aspecto visual y el desempeño del escritorio, a los requerimientos del sistema y del usuario.
El gestor de ventanas predeterminado para trabajar con KDE es KWin, y este se encarga de dar toda la apariencia y el estilo a las ventanas desplegables del escritorio, siendo sumamente personalizable. Kwin ofrece una estética característica al sistema visual de Linux.
KDE es uno de los entornos de escritorio con mayor compatibilidad que se han diseñado para Linux, ya que ofrece un nivel de adaptación a cambios tan sofisticado, que es considerado como uno de los entornos de escritorio menos predeterminados que existen actualmente.



GNOME


¿Que es Gnome?

Cuando se trabaja en el sistema operativo Linux, es importante escoger (entre la gran variedad que existe), el entorno de escritorio que mejor se adapte al usuario final y que sobretodo, ofrezca el mejor rendimiento del equipo en el que se va instalar.
Entre los entornos de escritorio más populares para Linux, encontramos Gnome, un entorno práctico, ligero y sencillo de usar, siendo ampliamente utilizado en todo el mundo.



ENTORNO DE ESCRITORIO.

Los entornos de escritorio, son paquetes de software diseñados para interactuar de manera gráfica e intuitiva, con el sistema operativo. Para realizar su tarea, los entornos se soportan en elementos gráficos y comunes para cualquier usuario, como iconos, barras de tarea, ventanas y animaciones.



CONOCIENDOMA GNOME

Gnome, es un tipo de entorno de escritorio, que además es un completo sistema de desarrollo para los sistemas operativos basados en Unix, como por ejemplo, Linux, FreeBDS y Solaris.
Gnome, nace como una alternativa completamente libre al famoso KDE. El proyecto Gnome, fue iniciado por los desarrolladores mexicanos Federico Mena y Miguel de Icaza y actualmente se encuentra disponible en un poco más de 48 idiomas, lo que aumenta considerablemente su versatilidad y lo hace muy atractivo a la vez.
Una de las principales características que podemos destacar de este entorno de escritorio, es su sistema interactivo completamente intuitivo, el cual ayuda al usuario trabajar de manera sencilla, simple y veloz.
El sistema de menú de Gnome, es bastante completo y su interfaz de usuario es altamente configurable, por lo que el usuario puede adaptar el funcionamiento de Gnome a las características de hardware de su equipo, maximizando así el uso de recursos de máquina.
El gestor de ventanas de Gnome, es bastante eficiente y muy adaptable, lográndose una gran facilidad de uso al trabajar con ventanas. Otra característica destacable de este entorno de escritorio, es su enorme cantidad de herramientas y aplicaciones, entre las que podemos encontrar procesadores de texto, hojas de cálculo, programas de mensajería, herramientas de programación, juegos, entre otros, todo lo cual convierte a Gnome en uno de los mejores entornos de escritorio que existen para Linux.
